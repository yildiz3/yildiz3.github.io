E-Commerce-Daten und Kundenbindungsanalyse mit SQL

Eine E-Commerce-Organisation erfordert eine gewisse Analyse der Verkaufs- und Lieferprozesse. So hofft die Organisation, die Chancen und Risiken für die Zukunft leichter vorhersagen zu können. Gemäß diesem Szenario werden Sie gebeten, die folgenden Analysen gemäß den gegebenen Anweisungen durchzuführen.

Projektanforderungen

Einleitung:

  • Sie müssen eine Datenbank erstellen und in die angegebenen csv-Dateien importieren. (Sie sollten recherchieren, wie Sie eine .csv-Datei oder eine .xlsx-Datei importieren.)
  • Während des Importvorgangs müssen Sie die Datumsspalten anpassen. Sie müssen die Datentypen und wie sie sein sollten, sorgfältig beobachten.
  • Die Daten sind nicht sehr sauber und vollständig normalisiert. Sie hindern Sie jedoch nicht daran, die angegebenen Aufgaben auszuführen. In einigen Fällen müssen Sie möglicherweise die Zeichenfolgen-, Fenster-, System- oder Datumsfunktionen verwenden.
  • Es kann Probleme geben, bei denen Sie die Tabellen aktualisieren müssen.
  • Überprüfen Sie manuell die Genauigkeit Ihrer Analyse.

Fähigkeiten benötigt

SQL
MYSQL
Microsoft Excel
