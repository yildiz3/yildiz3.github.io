Sentiment Analysis and Classification Project

Willkommen beim "Sentiment Analysis and Classification Project",diese Analyse konzentriert sich auf die Verwendung von Techniken der natürlichen Sprache, um allgemeine Trends in den schriftlichen Gedanken der Kunden zu finden. Das Ziel dieses Projekts ist es, anhand der Informationen in ihrem Bewertungstext vorherzusagen, ob Kunden das gekaufte Produkt weiterempfehlen.

Projektanforderungen

Das grundlegende Ziel dieses Projekts besteht darin, anhand der Informationen in ihrem Bewertungstext vorherzusagen, ob Kunden das von ihnen gekaufte Produkt weiterempfehlen.

  • Insbesondere sollte beachtet werden, dass die Erwartung in diesem Projekt darin besteht, nur die Variable "Review Text" zu verwenden und die anderen zu vernachlässigen.

Techniques

Major Tasks

  • Explorative Datenanalyse: Importieren Sie Module, laden Sie die Daten
  • Funktionsauswahl und Datenbereinigung: Funktionsauswahl und Spaltenname umbenennen, Erkennung fehlender Werte
  • Text-Mining: Tokenisierung, Rauschunterdrückung, Lexikon-Normalisierung
  • WordCloud - Wiederholung von Wörtern: Bewertungen erkennen, Wörter sammeln, Wortwolke erstellen
  • Stimmungsklassifikation mit maschinellem Lernen: Trainieren - Test-Split, Vektorisierung, TF-IDF, logistische Regression, Naive Bayes, Support Vector Machine, Random Forest, AdaBoost, Deep-Leraning-Modell, BERT-Modell, Modellvergleich

Completed using Jupyter Notebook.
